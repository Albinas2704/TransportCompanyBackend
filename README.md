## Project Description
This is the backend module of the Transport Company Management System.
It is developed using **Java Servlets**, **JDBC**, **MySQL**, and deployed on **Apache Tomcat**.

## Technologies Used
- Java (Servlets)
- JDBC
- MySQL
- Apache Tomcat 9
- HTML, CSS, JavaScript (Frontend)
- Git & GitHub

## Features
- User Login using Java Servlet
- Role-based access (Manager, Employee, Driver, Customer)
- Database connectivity using MySQL
- MVC-style project structure


## How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat
3. Update MySQL credentials in DB connection file
4. Run on server
5. Open: http://localhost:8080/TransportCompanyBackend/
